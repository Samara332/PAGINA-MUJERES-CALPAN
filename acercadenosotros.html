Acerca de nosotros

  Somos un conjunto de mujeres en Calpan dedicadas a diversas actividades económicas principalmente relacionadas con el campo y la agricultura, dentro de esta página encontrarás un espacio para realizar publicaciones que permitan contar tu experiencia con diferentes tipos de cultivo o temas de interés, adempás de fungir como una red de contacto especifica para tratar temas relacionados con las mujeres de Calpan.



  Este proyecto se realizó en el periodo de otoño 2021 dentro de la Universidad Iberoamericana Puebla con el apoyo del jardín etnobotánico de Cholula y se centró en ayudar a organizaciones de mujeres emprendedoras del municipio de Calpan cuya principal ocupación es el cultivo y comercialización de frutos como, por ejemplo: higo, tejocote, durazno, entre otros. Con el principal objetivo de crear un medio de comunicación entre las integrantes.



  Este proyecto busca que las mujeres tengan una herramienta de fácil acceso para poder dialogar y darse a conocer, siendo un espacio personalizado exclusivamente controlado por las mismas sin necesidad de depender de otras herramientas como lo pueden ser las redes sociales. Se reconoce el área tecnológica como una ventana de oportunidad para lograr aumentar la comunicación de las mujeres de Calpan así como para mostrar la identidad que une a todos los diferentes grupos que se encuentran dentro del proyecto.
